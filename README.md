# 液压缸选型设计工具 V2.3

基于《现代液压气动手册》(许仰曾/Steffen Haack, 2024) 的液压缸完整选型设计工具。

## 功能覆盖

9 步完整工程计算流程：

| 步骤 | 内容 | 参考标准 |
|------|------|----------|
| 1 | 工况与负载分析 | λ_p 负载率 |
| 2 | 系统压力等级与阀门通径匹配 | ISO 4401 / NG6-NG40 |
| 2a | 蓄能器选型（等温/绝热气体定律） | 《手册》第32章 |
| 3 | 缸径计算与标准化 | GB/T 2348 |
| 4 | 活塞杆直径与速比 | GB/T 2349 |
| 5 | 缸筒壁厚与材料强度 | 薄壁/Lamé 第四强度理论 |
| 6 | 活塞杆稳定性校核 | Euler/Tetmajer/屈服三区 |
| 7 | 密封系统与缓冲 | 温度退化/NAS 清洁度 |
| 8 | 流量计算与泵阀匹配 | 齿轮泵/叶片泵/柱塞泵 |
| 9 | 热平衡与散热器 | 风冷/水冷/板式换热器 |

## 材料数据库

9 种常用钢材: 45#, 27SiMn, 35#, Q235, Q345, 40Cr, 35CrMo, 38CrMoAl, ZG270-500

## 快速开始

**直接使用（无需 Python）：**
下载 `液压缸选型V2.3.exe`，双击运行。

**从源码运行：**
```bash
pip install -r requirements.txt  # 实际上仅需 Python 标准库 + tkinter
python main.py
```

## 运行测试

```bash
python tests/test_calculations.py    # 53 个单元测试
python tests/test_integration.py     # 端到端集成测试
```
